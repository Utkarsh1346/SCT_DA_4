# SCT_DA_4
📊 Marketing Campaign Analysis (EDA)
🔍 Introduction

This project focuses on performing Exploratory Data Analysis (EDA) on a marketing campaign dataset to evaluate the performance of different marketing channels and optimize budget allocation.

📁 Dataset

The dataset contains information about multiple marketing channels, including:

Impressions
Clicks
Conversions
Cost
ROI

🛠 Tools Used
Microsoft Excel
Pivot Tables
Data Cleaning Techniques
Data Visualization

📈 Analysis Performed
Cleaned and prepared raw data
Created key performance metrics (CTR, Conversion Rate)
Performed channel-wise analysis using pivot tables
Compared ROI, conversions, and engagement across channels

🔑 Key Insights
Email is the best-performing channel with highest ROI and conversions
Website shows lower ROI indicating inefficient budget usage
Other channels perform moderately and require optimization
Performance varies across channels, highlighting need for strategic allocation

💡 Recommendations
Increase budget allocation for Email marketing
Optimize or reduce spending on low-performing channels like Website
Improve conversion strategies for moderate channels
Continuously monitor performance for better decision-making

📌 Conclusion

This project demonstrates how data-driven analysis can help businesses optimize marketing strategies and improve ROI through effective channel selection.
